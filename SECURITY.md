# Security Policy

## Supported Versions

Currently, we only provide security updates for the latest version of Eko.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within Eko, please do **not** open a public issue. Instead, please report it via the following method:

1. **Email:** Send an email to the project maintainer (see GitHub profile).
2. **Details:** Include a description of the vulnerability, steps to reproduce, and potential impact.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
